# PlaywrightNew

Simple Playwright learning repository. It contains basic example scripts under `Chapter_01_Basics` and a sample Playwright test in `tests/`.

Quick start

1. Install dependencies:
```bash
npm install
npx playwright install
```

2. Run tests:
```bash
npm test
```

Files
- `Chapter_01_Basics/` — example JS snippets (learning material).
- `tests/` — Playwright test examples.

CI
- A GitHub Actions workflow is included at `.github/workflows/playwright.yml` to run Playwright tests on push and PR.
